Crie um site institucional completo para a PRONTA Engenharia usando React + Vite + Tailwind CSS.

Leia o arquivo briefing.md antes de começar.

Requisitos técnicos:
- Site one page
- Responsivo para celular, tablet e desktop
- Componentes organizados
- Código limpo e fácil de editar
- Dados de serviços, setores, diferenciais e números em arrays
- Sem backend
- CTA principal para WhatsApp
- Usar ícones da biblioteca lucide-react
- Usar animações leves com CSS/Tailwind, sem exagero
- Otimizar para carregamento rápido
- Criar estrutura profissional de pastas

Visual:
Industrial, técnico, robusto e moderno.
Usar paleta com azul petróleo, grafite, branco, cinza e detalhe em laranja/âmbar.

Conteúdo:
Usar os textos e informações do briefing.md.
